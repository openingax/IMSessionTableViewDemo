# IMSessionTableViewDemo
一个基于 UITablewView 的简单会话页面，主要展示加载会话消息时的动画处理。

#### 功能
1. 首次进入会话时，消息满屏与不满屏的处理（代码里改 msgCount 去观察）；
2. 下拉加载更多消息时的动画处理。

#### 依赖的第三方
下拉刷新控件：MJRefresh
